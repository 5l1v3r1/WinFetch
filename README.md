# Usage
Execute in **command prompt** (*must be in the same directory*):
> src/winfetch.bat

# Collected informations
1. *OS*
2. *OS Version*
3. *Uptime*
4. *Shell*
5. *Resolution*
6. *OS Type*
7. *System Drive*
8. *CPU Name*
9. *CPU Cores*
10. *CPU Type*
